    Los macrodatos, ​ también llamados datos masivos, inteligencia de datos, datos a gran escala o big data es un término que hace referencia a conjuntos de datos tan grandes y complejos que precisan de aplicaciones informáticas no tradicionales de procesamiento de datos para tratarlos adecuadamente. Tiene como concepto, la suma total de un conocimiento que tiene una forma de hacer las cosas, es decir, como se diseñan, como se producen, distribuyen y se venden los bienes o servicios. Es un conjunto de instrucciones o reglas definidas y no-ambiguas, ordenadas y finitas que permite, típicamente, solucionar un problema, realizar un cómputo, procesar datos y llevar a cabo otras tareas o actividades.​ Dados un estado inicial y una entrada, siguiendo los pasos sucesivos se llega a un estado final y se obtiene una solución.
